# Slide Memo

## 1. 슬라이드 메모 소개

화면 가장자리(왼쪽 또는 오른쪽)에 세로 탭으로 숨어 있다가, 클릭하면 슬라이드 애니메이션으로 펼쳐지는 Windows용 메모장.
다른 앱을 작업하면서 빠르게 메모를 작성하고 다시 화면 밖으로 숨길 수 있도록 설계되었습니다.

- Always-on-top 윈도우 (다른 앱 위에 항상 표시)
- 시스템 트레이 또는 작업표시줄에서 토글
- 메모마다 색상으로 구분된 세로 탭
- 모든 데이터는 로컬 SQLite에 저장 (외부 서버 없음)

---

## 2. 주요 기능

### 메모 작성/관리

- 메모 생성, 편집, 삭제, 검색, 정렬 (수정일 / 제목 / 생성일)
- 메모 고정 (우클릭) — 어떤 정렬에서도 항상 맨 위
- 휴지통 — 30일 후 자동 영구 삭제, 전체 비우기 버튼
- 자동 저장 (600ms 디바운싱)
- 클립보드 자동 감지 — 다른 앱에서 텍스트/이미지를 복사한 뒤 슬라이드 메모를 펼치면 "새 메모로 만들기" 배너 표시

### 서식

- 굵게 / 기울임 / 밑줄 / 취소선
- 글자색 (8색 + 사용자 정의)
- 배경색 (8색 + 해제)
- 글꼴/크기 (메모별로 다르게 지정 가능)
- 정렬 (왼쪽 / 가운데 / 오른쪽)
- 불릿 목록 / 번호 목록
- 링크 삽입 (URL)
- 표 삽입 (행/열/헤더 옵션)
- 날짜/시간 삽입 (메모장 포맷, ISO, 한국식 등 5종)
- 이미지 붙여넣기 (Ctrl+V)

### AI 기능 (옵션, 사용자 API 키 필요)

기본적으로 비활성. 설정 → AI 탭에서 활성화 + API 키 등록 후 사용.

- 요약 — 선택 영역 또는 메모 전체를 요약
- 번역 — 한↔영 자동 감지
- 맞춤법 교정 — 철자/문법/구두점 수정
- 제목 생성 — 본문 기반 짧은 제목 자동 생성
- 키워드 추출 — 5~10개 키워드를 본문 끝에 추가

지원 제공자:
- Anthropic Claude
- OpenAI (GPT)
- Google Gemini
- Ollama (로컬 모델, API 키 불필요)

### 음성 녹음 (옵션)

OpenAI Whisper API로 음성 → 텍스트 변환. 마이크 권한 + OpenAI 키 필요.

- 서식 툴바 마이크 버튼 클릭으로 녹음 시작/종료 토글
- 종료 시 자동으로 Whisper API 호출 → 변환된 텍스트가 새 메모로 생성
- 최대 녹음 시간 설정 (안전장치, 기본 5분)
- 녹음 파일 메모에 첨부 옵션

### 사용자화

- 좌/우 가장자리 선택
- 표시 방식 (트레이만 / 작업표시줄만 / 둘 다)
- 시스템 시작 시 자동 실행 (Windows / macOS / Linux)
- 인덱스 탭 가로 폭 (24~60 px)
- 메모 인덱스 탭 세로 길이 (60~200 px)
- 윈도우 폭/세로 크기 마우스 드래그로 조절
- 윈도우 세로 위치 상단 그립 드래그로 이동

## 3. 설치 및 실행

### A. 인스톨러로 설치 (권장, Windows)

배포된 `SlideMemo-Setup.exe`를 더블클릭하면 마법사 단계 없이 진행 바만 보였다가 자동으로 실행됩니다.

- 설치 위치: `%LOCALAPPDATA%\Programs\SlideMemo\` (UAC 프롬프트 없음, 관리자 권한 불필요)
- 시작 메뉴 + 바탕화면 바로가기 자동 생성
- 제거: 설정 → 앱 → 설치된 앱 → "Slide Memo" → 제거

### B. 소스에서 실행 (개발자)

필요한 것: Python 3.12 이상, [uv](https://docs.astral.sh/uv/)

```bash
uv sync
uv run python src/main.py
```

### 첫 실행

화면 가장자리(기본: 오른쪽)에 세로 탭이 표시되고, 메모가 없으면 빈 메모 1개가 자동 생성됩니다.

데이터 폴더 `~/.memo_slide/`가 자동으로 만들어집니다 (`~` = `C:\Users\<사용자명>`).

## 4-1. AI 기능 사용 (선택)

AI 기능은 기본 비활성화 상태이며, 사용하려면 본인 API 키가 필요합니다.

### 키 발급 페이지

| 제공자 | 발급 페이지 | 비고 |
|---|---|---|
| Anthropic | https://console.anthropic.com/ | 유료 |
| OpenAI | https://platform.openai.com/api-keys | 유료 |
| Google Gemini | https://aistudio.google.com/apikey | 무료 등급 있음 |
| Ollama | https://ollama.com/ | 로컬 설치, 무료 |

### 키 등록 방법

1. 트레이 아이콘 우클릭 → ⚙ 설정 → "AI" 탭
   (또는 펼친 메모장 우측 컬럼의 톱니바퀴 버튼)
2. "제공자" 드롭다운에서 사용할 서비스 선택
3. "API 키" 입력 → "테스트" 버튼으로 검증
4. "모델" 선택
5. "AI 기능 사용" 체크박스 켜기 → OK

키는 시스템의 `keyring` 보안 저장소에 저장되며, DB나 파일에 평문으로 남지 않습니다.

### 사용법

서식 툴바 아래 AI 버튼 5개 또는 단축키 (다음 섹션 참고).
본문에서 텍스트를 선택하고 누르면 그 영역에만, 선택 없이 누르면 메모 전체에 적용됩니다.

## 4-2. 음성 녹음 기능 사용

### 사전 준비

1. Windows 11 → 설정 → 개인정보 및 보안 → 마이크 → "앱이 마이크에 액세스 허용" ON
2. OpenAI API 키 (Whisper는 OpenAI 키 사용)
3. 설정 → AI 탭 하단 "음성 녹음 (Whisper)" 섹션:
   - "음성 녹음 기능 사용" 체크
   - "Whisper용 OpenAI API 키" 입력 → "테스트" 검증 (메인 AI 제공자가 OpenAI가 아니어도 Whisper만 따로 OpenAI 키 등록 가능)
4. (선택) "녹음 파일을 메모에 첨부" 체크 / "최대 녹음 시간" 조절

### 사용법

1. 서식 툴바의 마이크 버튼 클릭 → 녹음 시작 (버튼이 빨간색으로 변함)
2. 말하기
3. 다시 마이크 버튼 클릭 → 녹음 종료 + 자동 변환 시작
4. "✨ 음성 변환 중..." 표시가 사라지면 변환된 텍스트로 새 메모가 자동 생성됨

비용: Whisper API는 분당 약 $0.006 (약 8원).

## 5. 사용 방법

### 기본 조작

| 동작 | 방법 |
|---|---|
| 펼치기 | 메모 탭 클릭 / 트레이 아이콘 클릭 |
| 접기 | Esc / 같은 메모 탭 다시 클릭 |
| 다른 메모로 전환 | 다른 색상 탭 클릭 |
| 좌/우 가장자리 전환 | 트레이 메뉴 → "왼쪽 / 오른쪽 가장자리 전환" |
| 윈도우 폭 조절 | 본문 좌/우 가장자리 드래그 (펼친 상태) |
| 윈도우 세로 크기 조절 | 본문 위/아래 가장자리 드래그 (펼친 상태) |
| 윈도우 세로 위치 이동 | 상단 손 모양 그립 드래그 |

### 단축키 

| 단축키 | 동작 |
|---|---|
| `Esc` | 접기 (AI 진행 중이면 AI 취소) |
| `Ctrl+N` | 새 메모 |
| `Ctrl+F` | 검색창 포커스 |
| `Ctrl+S` | 즉시 저장 (자동 저장은 600ms 후) |
| `Ctrl+Shift+C` | 메모 본문 전체 클립보드로 복사 |
| `Ctrl+Shift+D` | 메모 삭제 — 일반 모드: 휴지통으로 / 휴지통 모드: 영구 삭제 |
| `Ctrl+B` / `Ctrl+I` / `Ctrl+U` | 굵게 / 기울임 / 밑줄 |
| `Ctrl+K` | 링크 삽입 (에디터 포커스 시) |
| `F5` | 윈도우 메모장 포맷 날짜 (`오후 3:24 2026-05-23`) |
| `Ctrl+;` | 오늘 날짜 (`2026-05-23`) |
| `Ctrl+Shift+;` | 현재 시간 (`15:24`) |
| `Ctrl+Alt+;` | ISO 형식 (`2026-05-23T15:24:00`) |
| `Ctrl+Shift+H` | 한국식 날짜 (`2026년 05월 23일 토요일`) |
| `Ctrl+Alt+S` | AI 요약 |
| `Ctrl+Alt+T` | AI 번역 |
| `Ctrl+Alt+P` | AI 맞춤법 교정 |
| `Ctrl+Alt+H` | AI 제목 생성 |
| `Ctrl+Alt+K` | AI 키워드 추출 |

### 메모 색상

서식 툴바와 별개로, 제목 입력 줄 오른쪽의 색상 점들로 메모 색을 바꿉니다.

- 그라데이션 2종 — sunrise (4색 대각선) / blossom (핑크→크림)
- 솔리드 7색 — ivory / blush / peach / cream / olive / lavender / mint
- 사용자 정의 hex 색상 (`+` 버튼)

### 설정 다이얼로그

트레이 메뉴 → ⚙ 설정 (또는 컬럼의 톱니바퀴 버튼). 세 탭:

- **일반** — 앱 위치(좌/우), 표시 방식, 인덱스 탭 폭/높이, 자동 실행, 클립보드 자동 감지
- **AI** — 제공자, 키, 모델, 토큰 사용량, 미리보기, 음성 녹음 옵션
- **정보** — 버전, 저작권, 데이터 폴더 열기

## 6. 데이터 저장 위치

모두 사용자 홈 디렉토리 아래에 저장됩니다 (`~` = `C:\Users\<사용자명>` 또는 macOS/Linux의 홈).

| 경로 | 내용 |
|---|---|
| `~/.memo_slide/memos.db` | SQLite 메모 DB (제목, 본문 HTML, 색상, 폰트, 핀, 휴지통 상태, 생성/수정 시각) |
| `~/.memo_slide/memos.db.bak` | DB 스키마 마이그레이션 시 자동 백업 (있을 때만) |
| `~/.memo_slide/images/img_*.png` | 메모에 붙여넣은 이미지 |
| `~/.memo_slide/temp/recording_*.wav` | 녹음 임시 파일 (변환 후 삭제) |
| `~/.memo_slide/audio/recording_*.wav` | "녹음 파일 첨부" 옵션 ON 시 보관 |
| 시스템 keyring | API 키 (Windows Credential Manager, macOS Keychain 등) |

### 백업 방법

`~/.memo_slide/` 폴더를 통째로 복사하면 모든 메모와 이미지가 보존됩니다. API 키는 시스템 keyring에 있으므로 별도 백업이 필요합니다 (재설치 시 다시 등록).

## 7. 문제 해결

### "AI 기능을 먼저 활성화하세요" 토스트가 뜹니다

설정 → AI 탭 → "AI 기능 사용" 체크박스가 꺼져 있습니다. 켜고 키도 등록되어 있어야 합니다.

### 마이크 버튼이 동작 안 합니다

다음을 순서대로 확인:
1. 설정 → AI 탭 → "AI 기능 사용" 체크
2. 같은 탭 하단 "음성 녹음 (Whisper)" 그룹 → "음성 녹음 기능 사용" 체크
3. "Whisper용 OpenAI API 키"에 OpenAI 키 등록 + 테스트 통과
4. Windows 11 → 설정 → 개인정보 및 보안 → 마이크에서 앱 권한 허용

### 클립보드 알림이 너무 자주 뜹니다

설정 → 일반 탭 → "메모 앱 펼칠 때 클립보드 자동 감지" 체크 해제.
또는 알림이 떴을 때 ✕ 버튼을 누르면 같은 내용은 다시 안 뜹니다 (현재 세션 한정).

### 앱 전체 폰트가 다르게 보입니다

앱 전역 폰트는 번들된 Pretendard를 사용합니다 (시스템 설치 불필요). 만약 다르게 보인다면 `assets/fonts/Pretendard-Regular.otf` / `Pretendard-Bold.otf`가 빠진 빌드일 가능성이 있어 OS 기본(Windows의 Segoe UI/Malgun Gothic)으로 fallback 된 상태입니다.

### DB가 손상된 것 같습니다

`~/.memo_slide/memos.db.bak`이 있으면 그것을 `memos.db`로 이름 바꾸어 복원할 수 있습니다 (마이그레이션 시 자동 생성된 백업).

### `uv sync`가 OneDrive 폴더에서 실패합니다

OneDrive의 hardlink 제한 때문에 발생합니다. `uv cache clean` 후 다시 시도하세요.
